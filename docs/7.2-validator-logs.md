## How to Reproduce

```bash
# Requires Python 3.12 (bt SDK 10.x has issues with Python 3.14)
python3.12 -m venv btsdk_venv && source btsdk_venv/bin/activate
pip install bittensor && pip install -e .

# Start all miners (10 processes)
port=8091
for hotkey in default miner2 miner3 miner4 miner5 miner6 miner7 miner8 miner9 miner10; do
  nohup python neurons/miner.py \
    --netuid 26 --subtensor.network test \
    --wallet.name miner --wallet.hotkey $hotkey \
    --axon.port $port --logging.info > logs/miner-$hotkey.log 2>&1 &
  port=$((port + 1))
done

# Start validators (3 processes)
for pair in "default:8111" "vali2:8102" "vali3:8103"; do
  hotkey="${pair%%:*}"; port="${pair##*:}"
  nohup python neurons/validator.py \
    --netuid 26 --subtensor.network test \
    --wallet.name vali --wallet.hotkey $hotkey \
    --axon.port $port --logging.info > logs/vali-$hotkey.log 2>&1 &
done

# Monitor scoring state
cat ~/.bittensor/neurons/vali/default/netuid26/validator/score_aggregator.json | python3 -m json.tool

# Verify weights on chain
python -c "
import bittensor as bt
sub = bt.Subtensor(network='test')
for entry in sub.weights(netuid=26):
    print(f'Validator UID {entry[0]} -> {len(entry[1])} targets')
"

# Inspect audit flags written to disk by the validator (path is configurable via CSWON_AUDIT_LOG_PATH)
# Default location: ./audit_flags.jsonl (relative to the directory where the validator was started)
tail -f audit_flags.jsonl                       # live tail during demo
grep '"uid": 3' audit_flags.jsonl | python3 -m json.tool   # filter by miner UID

# Or query the in-memory HTTP endpoint (last 100 flags, survives only until restart)
curl -s http://127.0.0.1:9090/audit-flags | python3 -m json.tool
```

> **Audit flag persistence:** Flags are written to `audit_flags.jsonl` (newline-delimited JSON) and survive validator restarts. The path can be overridden with the `CSWON_AUDIT_LOG_PATH` environment variable. The in-memory HTTP endpoint at `127.0.0.1:9090/audit-flags` serves the last 100 flags for the current process only.
