## How to Reproduce

```bash
# Start validator and tail logs
python neurons/validator.py \
  --netuid 26 --subtensor.network test \
  --wallet.name vali --wallet.hotkey default \
  --axon.port 8101 2>&1 | tee validator.log

# Or with PM2 (recommended for persistent operation):
pm2 start neurons/validator.py --name "vali-default" --interpreter python -- \
  --netuid 26 --subtensor.network test \
  --wallet.name vali --wallet.hotkey default --axon.port 8101
pm2 logs vali-default
```
