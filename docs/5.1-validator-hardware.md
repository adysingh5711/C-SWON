# 5.1 Hardware Requirements

Validators run sandboxed Docker containers, automated test runners, and ROUGE-L scoring.

| Requirement | Minimum | Recommended |
|---|---|---|
| CPU | 8 cores | 16 cores |
| RAM | 32 GB | 64 GB |
| SSD storage | 500 GB | 1 TB NVMe |
| Network | 1 Gbps | 10 Gbps |
| Docker | 24.x+ | 26.x+ |
| Python | 3.10+ | 3.11+ |
| Uptime target | 95% | 99.5% |

## Subnet Call Authentication

| Stage | Authentication Model |
|---|---|
| Testnet | Mock execution (`CSWON_MOCK_EXEC=true`). No real calls. |
| Mainnet bootstrap | Dedicated hotkey on each partner subnet. Standard rates. |
| Mainnet at scale | Negotiated API-tier access. Revenue-share with partners. |
