# 6.3 Running Locally (Staging)

This guide walks you through running C-SWON on a **local blockchain** — not connected to testnet or mainnet. This is ideal for development and testing.

> **Local blockchain vs local subtensor node:** A local blockchain spins up isolated authority nodes. A local subtensor node connects to and syncs with mainnet. This guide is for the former.

## Prerequisites

- [Bittensor](https://github.com/opentensor/bittensor#install) installed
- Docker (for the local chain container)

## 1. Install Substrate Dependencies

```bash
sudo apt update
sudo apt install --assume-yes make build-essential git clang curl libssl-dev llvm libudev-dev protobuf-compiler
```

## 2. Install Rust and Cargo

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source "$HOME/.cargo/env"
```

## 3. Clone Subtensor

```bash
git clone https://github.com/opentensor/subtensor.git
```

## 4. Setup Rust

```bash
./subtensor/scripts/init.sh
```

## 5. Initialize Local Chain

```bash
cargo build -p node-subtensor --profile production --features pow-faucet
BUILD_BINARY=0 ./scripts/localnet.sh
```

## 6. Install C-SWON

```bash
git clone https://github.com/adysingh5711/C-SWON.git
cd C-SWON
python -m pip install -e .
```

## 7. Set Up Wallets

```bash
btcli wallet new_coldkey --wallet.name owner
btcli wallet new_coldkey --wallet.name miner
btcli wallet new_hotkey --wallet.name miner --wallet.hotkey default
btcli wallet new_coldkey --wallet.name validator
btcli wallet new_hotkey --wallet.name validator --wallet.hotkey default
```

## 8. Mint Tokens

```bash
btcli wallet faucet --wallet.name owner --subtensor.chain_endpoint ws://127.0.0.1:9946
btcli wallet faucet --wallet.name validator --subtensor.chain_endpoint ws://127.0.0.1:9946
```

## 9. Create Subnet

```bash
btcli subnet create --wallet.name owner --subtensor.chain_endpoint ws://127.0.0.1:9946
```

## 10. Register Keys

```bash
# Miner
btcli subnet register --wallet.name miner --wallet.hotkey default \
  --subtensor.chain_endpoint ws://127.0.0.1:9946

# Validator
btcli subnet register --wallet.name validator --wallet.hotkey default \
  --subtensor.chain_endpoint ws://127.0.0.1:9946
```

## 11. Add Stake

```bash
btcli stake add --wallet.name validator --wallet.hotkey default \
  --subtensor.chain_endpoint ws://127.0.0.1:9946
```

## 12. Verify Registrations

```bash
btcli subnet list --subtensor.chain_endpoint ws://127.0.0.1:9946
```

## 13. Run Miner and Validator

```bash
# Set environment
export CSWON_MOCK_EXEC=true
export CSWON_SYNTHETIC_SALT=$(python -c "import secrets; print(secrets.token_hex(32))")

# Miner
python neurons/miner.py --netuid 1 \
  --subtensor.chain_endpoint ws://127.0.0.1:9946 \
  --wallet.name miner --wallet.hotkey default --logging.debug

# Validator (separate terminal)
python neurons/validator.py --netuid 1 \
  --subtensor.chain_endpoint ws://127.0.0.1:9946 \
  --wallet.name validator --wallet.hotkey default --logging.debug
```

## 14. Verify Incentive Mechanism

```bash
btcli wallet overview --wallet.name miner --subtensor.chain_endpoint ws://127.0.0.1:9946
```

After a subnet tempo elapses (360 blocks), you should see TAO distribution beginning.

## Ending Your Session

Press `CTRL + C` in the terminal to stop nodes.
